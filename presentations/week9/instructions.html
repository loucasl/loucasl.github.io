One Page Design

CONCEPT

A client that owns a restaurant has come to you looking for your website company to redesign the ONE PAGE layout of their current menu page.
Their customers are disinterested by the design of their menu and because of this, not many people want to come to their restaurant. They have provided you with the basic outline which includes the HTML of the site. They are looking for you to redesign it to be more appealing to customers.

THE ACTIVITY

  • Brainstorm how YOU want the layout to look!
  • This does NOT have to be handed in. It is for your own brainstorming
  • Think of a colour scheme, typography and overall theme you want your website to have.

THE TASK

  • Go to this LINK and download the raw upload.html file
  • Add the file to your presentations/week9 folder in Atom.
  • Using the HTML already provided, style the site.
  • Feel free to make changes that you see fit to make the website better, without changing the menu items.
  • Using the classes already provided to you, link to the different parts of the page. By doing this you are ensuring that your site is a ONE PAGE DESIGN

REQUIREMENTS

  1. Use page anchors to jump to different parts of the page.
  2. Have a cohesive theme.
  3. Place your name in the title.
  4. Keep the content of the HTML the same.
  5. Include at least two images.
  6. Be creative & have fun!

Additions if wanted!

  • If you'd like you can add JavaScript to the page to give it additional functonality.

UPLOAD

  • When complete, link the file to yourmain index.html page.
  • Then commit and push to origin.

RUBRIC

  • Use at LEAST two images - 0/2
  • Overall Appearance and Aesthetic - 0/5
  • Page Anchors and Functionality - 0/3
  • Kept majority of the content the same - 0/3
  • BONUS - Added some elements of Javascript - 0/2
  • TOTAL: 0/15
